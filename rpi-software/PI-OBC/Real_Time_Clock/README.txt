RTC Driver for Raspberry Pi:

- Created for the MCP79410, Data Sheet: http://ww1.microchip.com/downloads/en/devicedoc/20002266h.pdf
- Driver uses I2C communication
- Functionality:
    -Enable/Disable Internal Oscillator (Start and Stop Clock)
    -Get Date and Time from Clock
    -Enable/Disable Backup Battery
    -Reset Clock 
- More Documentation to follow
